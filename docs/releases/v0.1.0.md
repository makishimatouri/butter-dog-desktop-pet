# v0.1.0

发布日期：2026-07-14

`v0.1.0` 是 Butter Dog Trash Pet 的首个可用版本。这个版本把 Codex 黄油小狗图集封装成独立 Electron 桌面应用，并加入文件投喂到系统回收站的功能。

## 主要功能

- 文件或文件夹拖到小狗身上后，移入系统回收站。
- 投喂时播放简洁等待动作，放开后播放吃掉动作。
- 鼠标悬停时随机播放招手或举手蹦跳回应。
- 拖动小狗时根据方向播放左右跑步动作。
- 右键菜单提供打开回收站、调整大小、退出。
- 大小设置会保存到本机，下次打开继续使用。

## 平台说明

- Windows 使用 Electron 的系统回收站能力，目标行为是进入 Recycle Bin，不做永久删除。
- macOS 使用系统 Trash。
- Linux 保留 `trash:///` 打开入口，但此版本未作为主要验证平台。

## 打包产物

本地打包命令：

```bash
npm run package:mac
npm run package:mac:arm64
npm run package:win
```

GitHub Actions 会在 `main` 分支推送后构建 Windows x64 包。

