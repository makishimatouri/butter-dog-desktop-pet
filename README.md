# Butter Dog Desktop Pet

Portable Windows desktop pet for Butter Dog.

## Local Development

```bash
npm install
npm start
```

## Build Windows Portable Package

```bash
npm run dist:win
```

The GitHub Actions workflow builds a portable Windows release artifact from `windows-latest`.

## Controls

- Drag the pet to move it.
- Double-click to make it jump.
- Right-click for the menu.
- Click the tray icon to show or hide.
